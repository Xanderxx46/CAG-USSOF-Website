Background Info

        As Machine Gunners, we are in charge of keeping the enemies heads down. We will be providing suppressive fire to make the enimies not wanna peak their heads up.

        We will be the main ones on any vehicles with Machine Guns on them, which means we not only have to know how to use a 5.56 mm machine gun with a scope, but also .50 CAL (and 7.62mm) ones without a scope.

        The main MG we will be using is a m249 or m240. Both are hand held and can be used in the field no matter where we are. (Ask Xander for any other authorized machine guns

Procedures

  1. Locate the enemy
  2. State which direction you will be suppressing
  3. Check in front of you, ensure no friendly is there
  4. Shoot only 2-3 bullets at a time, spreading out near where the enemy is
